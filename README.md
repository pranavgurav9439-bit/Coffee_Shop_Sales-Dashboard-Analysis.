# Coffee Shop Sales Dashboard

## Project Description

This is an interactive **Excel Coffee Shop Sales Dashboard** created to analyze coffee shop transaction data. The dashboard is built using **Pivot Tables, Pivot Charts, KPI Cards, and Slicers** to convert raw sales data into meaningful business insights.

The dashboard helps understand total sales, customer footfall, average bill value, average quantity per order, product performance, store-wise sales, peak ordering hours, product category contribution, and weekday sales trends.

---

## Dashboard Overview

The dashboard contains four main KPI cards:

* **Total Sales** – Shows total revenue generated from all transactions.
* **Total Footfall** – Shows total number of transactions/orders.
* **Avg Bill/Person** – Shows average bill amount per transaction.
* **Avg Qty/Order** – Shows average quantity ordered per transaction.

---

## Charts Used

The dashboard includes the following charts:

1. **Quantity Ordered by Hour**
   Shows peak ordering hours of the coffee shop.

2. **Sales Distribution by Category**
   Shows which product category contributes the most to sales.

3. **Order Distribution by Size**
   Shows customer preference for Small, Regular, Large, and Not Defined sizes.

4. **Store-wise Sales and Footfall**
   Compares sales and footfall across different store locations.

5. **Top 5 Products by Sales**
   Shows the highest revenue-generating products.

6. **Sales by Weekday**
   Shows sales performance from Monday to Sunday.

---

## Slicers Used

The dashboard has three slicers:

* **Month Name**
* **Day Name**
* **Store Location**

These slicers make the dashboard interactive and allow users to filter the data easily.
